Sobre el proyecto
---------------
El proyecto está estructurado de la siguiente manera:

-> Proyecto
	'-> bin/		#Carpeta donde la herramienta make guardará los archivos binarios.
	'-> headers/	#Carpeta donde se encuentran todos los headers de nuestra proyecto.
	'-> src/		#Carpeta donde se encuentran los archivos fuente (.cpp) de nuestro proyecto.
	'-> makefile	#Este archivo es el que utiliza la herramienta make para compilar.

Las especificaciones de las técnicas de la implementación, así como el código se proporcionan en archivos por separado.


Sobre el programa
---------------------
+ Lenguaje: C++
+ Compilador: g++-4.4 (IMPORTANTE!! Lo intentamos compilar con g++-4.6 y parecía no "linkear" adecuadamente)
+ Dependencias: GTK+2.0
+ Plataforma de desarrollo: Ubuntu/Linux

Instrucciones para ejecutar
----------------------------
No requiere ningún parámetro, simplemente se invoca el archivo ejecutable resultande de la compilación.

Ejecutable: bin/xcarjiProyecto1
